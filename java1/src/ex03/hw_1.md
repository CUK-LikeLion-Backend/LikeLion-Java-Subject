# Q.  Java에서 함수(function)를 선언할 때 사용되는 키워드(keyword)는 무엇인가요?
## 메서드 구조

    리턴자료형 메서드명(입력자료형1 매개변수1, 입력자료형2 매개변수2, ...) {
    ...    

    return 리턴값;  // 리턴자료형이 void 인 경우에는 return 문이 필요없다.}
## 아래 설명될 키워드들은 접근 지정자 키워드이다.
## 접근지정자 
### 메서드에 접근할 수 있는 범위를 지정 키워드
###  1. public : 모든 곳에서 접근 가능
###  2. private : 자기 자신 클래스에서만 접근 가능
###  3. protected : 자기 자신 클래스와 상속된 클래스에서 접근 가능
###  4. default(package-private) : 	자기 자신 클래스와 상속된 클래스에서 접근 가능
#### 접근 허용 가능 범위 순위 : public > protected > default > private
### 메서드의 특성을 지정 키워드
###  1. static : 클래스의, 공통적인
###            - 클래스 변수 static은 인스턴스에 관계없이 같은 값을 갖는다.
###            - 메모리에 고정적으로 할당되어, 프로그램이 종료될 때 해제되는 변수 
###            - 인스턴스를 생성하지 않고도 사용 가능
###  2. final : 마지막의, 변경될 수 없는
###           - 메서드에 사용되면 오버라이딩, 변수는 변경할수없는 상수, 클래스는 자신을 확장하는 자손 클래스를 정의하지 못하게 된다.
###  3. abstract : 미완성이라는 뜻으로 메서드의 선언부만 작성하고 실제 수행내용은 구현하지 않은 추상메서드를 선언하는데 사용
###  4. synchronized :  thread의 동기화를 의한 메서드이다.
