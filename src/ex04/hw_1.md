# 자바 연산자

> ### 산술 연산자
| 연산  | 연산자 | 우선순위 |
|-----|-----|------|
| 덧셈  | +   | 1    |
| 뺄셈  | -   | 1    |
| 곱셈  | *   | 0    |
| 나눗셈 | /   | 0    |
| 나머지 | %   | 0    |

> ### 논리 연산자
| 연산         | 연산자 | 우선순위 |
|------------|----|------|
| and(논리곱)   | && | 3    |
| 비트 단위 논리연산 | &  | 1    |
| or(논리합)    | l l | 4    |
| xor(배타적논리합) | ^  | 2    |
| not(부정)    | !  | 0    |

> ### 비교 연산자
| 연산자 | 우선순위 |
|-----|------|
| \>  | 0    |
| <   | 0    |
| \>= | 0    |
| <=  | 0    |
| ==  | 1    |
| !=  | 1    |
